# Module 08: Classroom Course Player

## Route
- `/classroom/[courseId]`

## Goal
- Distraction-free learning view for a single course.

## Required UX Blocks
- Back + breadcrumbs
- Main player area (`Start Class`, `Open on YouTube`)
- Lesson/course navigator
- Tabs: class info, related, comments, materials/resources

## Current Data Sources
- `courses`, `lessons`, `resources`
- `user_progress`
- `lesson_reactions`
- `lesson_summaries`
- `lesson_comments`, `lesson_comment_reactions`

## Related Features
- Like/dislike counts
- AI summary
- Related video/content list
- Next/previous course navigation

## Admin Ownership
- Lesson sequencing and resource quality.
- Course materials/articles mapping from academic plan.
