# Module 06: Student Dashboard

## Route
- `/dashboard`

## Goal
- Show returning student snapshot and next best actions.

## Main Sections
- Greeting
- KPI cards (GPA, credits, passed, in-progress)
- Recent activity
- Quick links to courses/progress

## Data Sources
- `user_progress`
- `course_grades`
- `cumulative_gpas`
- enrolled course records

## Admin Ownership
- KPI definitions and dashboard card labels.
