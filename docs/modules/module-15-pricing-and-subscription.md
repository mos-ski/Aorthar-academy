# Module 15: Pricing and Subscription

## Route
- `/pricing`

## Goal
- Convert free students to paid plans and unlock premium tracks.

## API + Integrations
- `POST /api/payments/checkout`
- `POST /api/webhooks/paystack`

## Data Sources
- `plans`
- `subscriptions`
- `transactions`
- `webhook_events`

## Admin Ownership
- Plan catalog management.
- Pricing updates and campaign offers.
- Reconciliation for failed/successful payments.
