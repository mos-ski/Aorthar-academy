**Last Updated:** 2026-04-03

## Overview

Each department has its own 4-year curriculum (Year 100 → 400, 2 semesters each, 3-5 courses per semester). Courses within each semester have 3-5 classes (lessons).

## Curriculum Structure

```
Department (e.g. Product Design)
  └── Year 100 (Design Foundations)
        ├── Semester 1
        │     ├── DES101 — Introduction to Visual Design (5 classes)
        │     ├── DES103 — Typography Fundamentals (5 classes)
        │     ├── DES105 — Color Theory for Designers (5 classes)
        │     ├── DES107 — Introduction to Design Software (5 classes)
        │     └── PM101 — Introduction to Design Thinking (5 classes)
        └── Semester 2
              ├── DES102 — UI Design Fundamentals (5 classes)
              ├── DES104 — Introduction to UX Design (5 classes)
              ├── DES106 — Design Psychology & Human Behavior (5 classes)
              ├── DEV102 — HTML & CSS for Designers (5 classes)
              └── UXR102 — Fundamentals of User Research (5 classes)
  └── Year 200 (Intermediate Craft)
        └── ...
  └── Year 300 (Advanced Systems & Strategy)
        └── ...
  └── Year 400 (Professional Practice & Leadership)
        └── ...
```

## Course Code Convention

| Prefix | Department |
|--------|------------|
| `DES` | Design (general) |
| `PM` | Product Management |
| `DEV` | Development |
| `UXR` | User Research |
| `PORT` | Portfolio |
| `INT` | Interview |
| `FRE` | Freelancing |
| `CAP` | Capstone |
| `ETH` | Ethics |
| `TEAM` | Team & Leadership |

Course code format: `[PREFIX][LEVEL]` (e.g. `DES101`, `PM302`)

---

## Class Structure

Each course has 3-5 classes (lessons). Each class is a single learning unit with:

- **Title**: Topic name (e.g. "How to Choose the Right Color System")
- **YouTube Resource**: Embedded video link
- **Optional Markdown Notes**: Supplementary text
