1. What is the Linux kernel? — **Answer:** The core of the Linux operating system — managing hardware resources, processes, memory, and system calls

*Quiz (5 questions):*
1. What is a container? — **Answer:** A lightweight, standalone, executable package that includes everything needed to run an application — code, runtime, libraries, and settings
2. What is the difference between a container and a virtual machine? — **Answer:** Containers share the host OS kernel (lighter, faster); VMs have their own OS (heavier, more isolated)
3. What is a Dockerfile? — **Answer:** A text file with instructions for building a Docker image — FROM, RUN, COPY, CMD, etc.
4. What is Docker Compose? — **Answer:** A tool for defining and running multi-container applications — using a YAML file to configure services, networks, and volumes
5. What is a multi-stage build? — **Answer:** Using multiple FROM statements in a Dockerfile — building in one stage and copying only the final artifacts to a smaller production image

*Quiz (5 questions):*
1. What do interviewers look for in DevOps candidates? — **Answer:** Technical breadth, problem-solving ability, automation mindset, collaboration skills, and continuous learning
2. How should you approach a DevOps system design question? — **Answer:** Clarify requirements, propose architecture, explain tradeoffs, discuss scalability, and address security
3. What is the most important DevOps skill? — **Answer:** Problem-solving — the ability to diagnose issues, find root causes, and implement solutions
4. How do you demonstrate DevOps experience in an interview? — **Answer:** Through specific examples — describing pipelines you built, incidents you resolved, and improvements you made
5. What is the biggest challenge in DevOps interviews? — **Answer:** The breadth of knowledge required — from Linux to Kubernetes to CI/CD to security

*Related Articles:*

- Data Privacy — https://ndpr.gov.ng/
- Security Responsibility — https://owasp.org/www-project-top-ten/

*Quiz (5 questions):*
1. What is the ethical responsibility of a DevOps engineer? — **Answer:** To protect user data, maintain security, ensure reliability, and act with integrity
2. What is the biggest ethical risk in DevOps? — **Answer:** Exposing sensitive data through misconfigured infrastructure, exposed secrets, or inadequate access controls
3. What is the ethical test for automation? — **Answer:** Would you be comfortable if this automation made a decision about you or someone you love?
4. How should a DevOps engineer handle a security vulnerability? — **Answer:** Report it immediately, assess impact, implement fix, communicate with stakeholders, and document the incident
5. Why is access control important in DevOps? — **Answer:** Because DevOps engineers have access to production systems — and unrestricted access can lead to accidental or intentional harm
