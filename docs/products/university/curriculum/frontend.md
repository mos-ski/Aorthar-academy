## YEAR 100 — FE FOUNDATIONS

**Goal:** Produce graduates who bridge the gap between design and engineering. Expert developers who can build beautiful, accessible, and high-performance user interfaces from design mockups.

### Semester 1

**DEV101 — JavaScript Fundamentals & ES6+** *(3 Credits)*
*Description:* Comprehensive introduction to JavaScript — the language of the web. From basic syntax to modern ES6+ features.
*Classes:*
1. Intro to JS & Core Concepts — Variables, data types
2. Operators & Type Coercion — `==` vs `===`
3. Control Flow: Conditionals — `if/else`, ternary, `switch`
4. Control Flow: Loops — `for`, `while`, `for...of`
5. Functions — Declarations, parameters, return values
6. Scope & Closures — Global vs local, lexical scope
7. Arrays & Array Methods — `push`, `pop`, `map`, `filter`
8. Objects — Literals, properties, `this` keyword
9. ES6+: Arrow Functions & Template Literals
10. ES6+: Destructuring & Spread/Rest
11. Asynchronous JS: Callbacks & Promises
12. Asynchronous JS: Async/Await, error handling

**DES101 — Introduction to Visual Design** *(3 Credits)*
*Description:* Foundational visual design principles every frontend engineer should understand.
*See: [Product Design — DES101](./product-design.md)*

**PM103 — Introduction to Agile & Scrum** *(3 Credits)*
*Description:* Core Agile and Scrum principles for working in product teams.

**DEV103 — Data Structures & Algorithms in JS** *(3 Credits)*
*Description:* Essential CS concepts implemented in JavaScript — arrays, linked lists, trees, sorting, searching.

**CS101 — Computer Science Fundamentals** *(3 Credits)*
*Description:* Core CS concepts — binary, memory, time/space complexity, basic algorithms.

### Semester 2

**DEV102 — Advanced HTML & CSS** *(3 Credits)*
*Description:* Master semantic HTML, CSS Grid, Flexbox, custom properties, and responsive design patterns.

**DES102 — UI Design Fundamentals** *(3 Credits)*
*Description:* Understand UI design principles so you can implement designs accurately and suggest improvements.

**DEV104 — Introduction to React & Component Architecture** *(3 Credits)*
*Description:* Learn React fundamentals — components, props, state, JSX, and component composition.

**DEV106 — Browser APIs & The DOM** *(3 Credits)*
*Description:* Understand the DOM, browser rendering, events, and browser APIs (storage, fetch, geolocation).

**QA102 — Introduction to Testing** *(3 Credits)*
*Description:* Unit testing, integration testing, and test-driven development fundamentals.

## YEAR 200 — INTERMEDIATE FRONTEND

### Semester 1

**DEV201 — Advanced React (Hooks & Context)** *(3 Credits)*
*Description:* Master React Hooks, Context API, custom hooks, and advanced component patterns.

**DEV203 — State Management (Redux, Zustand)** *(3 Credits)*
*Description:* Learn state management patterns — Redux, Zustand, Jotai, and when to use each.

**DEV205 — Component Libraries & Storybook** *(3 Credits)*
*Description:* Build reusable component libraries with Storybook for documentation and visual testing.

**QA201 — Frontend Testing (Jest, React Testing Library)** *(3 Credits)*
*Description:* Advanced testing strategies for React applications — unit, integration, and E2E tests.

**DES201 — Advanced UI & Layout Systems** *(3 Credits)*
*Description:* Complex layout systems, responsive design patterns, and CSS architecture at scale.

### Semester 2

**DEV202 — TypeScript Fundamentals** *(3 Credits)*
*Description:* Add type safety to JavaScript — interfaces, generics, utility types, and TypeScript with React.

**DEV204 — Web Performance & Optimization** *(3 Credits)*
*Description:* Core Web Vitals, lazy loading, code splitting, bundle analysis, and rendering optimization.

**DEV206 — Build Tools (Webpack, Vite)** *(3 Credits)*
*Description:* Configure and optimize build tools for modern frontend applications.

**DEV208 — Advanced CSS & Animations** *(3 Credits)*
*Description:* CSS animations, transitions, keyframes, and creating delightful micro-interactions.

**API201 — Consuming REST & GraphQL APIs** *(3 Credits)*
*Description:* Fetch data from REST and GraphQL APIs — error handling, caching, and data normalization.

## YEAR 300 — ADVANCED FRONTEND & SYSTEMS

### Semester 1

**DEV301 — Frontend System Design** *(3 Credits)*
*Description:* Design scalable frontend architectures — module federation, micro-frontends, and shared libraries.

**DEV303 — Accessibility Engineering (A11y)** *(3 Credits)*
*Description:* Build accessible web applications — WCAG compliance, screen reader testing, keyboard navigation.

**DEV305 — GraphQL Clients (Apollo, Relay)** *(3 Credits)*
*Description:* Advanced GraphQL patterns — caching, optimistic updates, subscriptions, and code generation.

**SEC301 — Frontend Security (OWASP)** *(3 Credits)*
*Description:* XSS, CSRF, CSP, secure coding practices, and security best practices for frontend applications.

**DEV307 — Advanced TypeScript** *(3 Credits)*
*Description:* Advanced type patterns — conditional types, mapped types, template literals, and type utilities.

### Semester 2

**DEV302 — Server-Side Rendering (Next.js)** *(3 Credits)*
*Description:* Build SSR/SSG applications with Next.js — routing, data fetching, API routes, and middleware.

**DEV304 — Progressive Web Apps (PWAs)** *(3 Credits)*
*Description:* Build installable, offline-capable web apps — service workers, app manifests, and push notifications.

**DEV306 — Real-time Applications (WebSockets, WebRTC)** *(3 Credits)*
*Description:* Build real-time features — chat, live updates, video calls, and collaborative editing.

**DEV308 — Introduction to Web Assembly (WASM)** *(3 Credits)*
*Description:* Run high-performance code in the browser — WASM basics, Rust to WASM, and use cases.

**CI301 — CI/CD for Frontend (GitHub Actions)** *(3 Credits)*
*Description:* Set up automated testing, linting, building, and deployment pipelines.

## YEAR 400 — PROFESSIONAL PRACTICE

### Semester 1

**INT402 — Mastering the Frontend Interview** *(3 Credits)*
*Description:* Prepare for frontend interviews — coding challenges, system design, React deep-dives, and behavioral rounds.

**PORT402 — Building Complex Frontend Applications** *(3 Credits)*
*Description:* Build a production-quality frontend application as a portfolio piece — full stack, tested, deployed.

**OSS401 — Contributing to Open Source** *(3 Credits)*
*Description:* Learn to contribute to open source projects — finding projects, making PRs, and building reputation.

**FRE402 — Freelancing for Developers** *(3 Credits)*
*Description:* Build a freelance development practice — finding clients, scoping, pricing, and delivering quality work.

**DEV401 — Specialization I** *(3 Credits)*
*Description:* Choose a specialization: Web3, WebGL/Three.js, Mobile (React Native), or Desktop (Electron).

### Semester 2

**CAP402 — Capstone Project: Frontend Application** *(6 Credits)*
*Description:* Build and ship a complex frontend application — from design to deployment with full testing.

**ETH402 — Ethics in Engineering** *(3 Credits)*
*Description:* Ethical considerations in software development — accessibility, privacy, bias, and responsible engineering.

**TEAM402 — Mentorship & Technical Leadership** *(3 Credits)*
*Description:* Develop mentorship skills and learn to lead technical teams, conduct code reviews, and drive technical decisions.
