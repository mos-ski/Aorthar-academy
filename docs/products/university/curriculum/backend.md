# Backend Engineering Curriculum

**Status:** ✅ Complete
**Last Updated:** 2026-04-03

## YEAR 100 — BE FOUNDATIONS

**Goal:** Produce graduates who can build and maintain server-side logic, databases, and APIs. Experts in scalability, security, and performance.

### Semester 1

**DEV105 — Introduction to Backend Development (Node.js)** *(3 Credits)*
*Description:* Server-side programming with Node.js — building applications outside the browser, handling HTTP requests, and setting up APIs.
*Classes:*
1. What is Backend? Intro to Node.js — Client vs Server, Blocking vs Non-Blocking I/O
2. Node.js Modules & `require` — CommonJS, `module.exports`, core modules
3. The File System (fs) Module — Reading/writing files, sync vs async
4. The HTTP Module — Creating a basic HTTP server, handling requests
5. Introduction to Express.js — Installing Express, setting up a basic server
6. Express.js: Routing — HTTP methods, route parameters
7. Express.js: Middleware — Custom middleware, `next()`
8. Handling Requests & Responses — `req`/`res` objects, JSON, status codes
9. Introduction to REST APIs — REST principles, statelessness, resource URLs
10. Building a Simple CRUD API — GET, POST, PUT, DELETE endpoints
11. Asynchronous Patterns in Node.js — Event loop, callbacks, Promises, Async/Await
12. Environment Variables — `.env` files, configuration, secrets

**DB101 — Database Fundamentals (SQL & NoSQL)** *(3 Credits)*
*Description:* Core database concepts — relational vs document databases, queries, indexing, and data modeling.

**PM103 — Introduction to Agile & Scrum** *(3 Credits)*
*Description:* Core Agile and Scrum principles for working in product teams.

**DEV103 — Data Structures & Algorithms** *(3 Credits)*
*Description:* Essential CS concepts — arrays, linked lists, trees, hash maps, sorting, searching, Big-O notation.

**CS101 — Computer Science Fundamentals** *(3 Credits)*
*Description:* Core CS concepts — binary, memory management, time/space complexity, basic algorithms.

### Semester 2

**DEV108 — Building RESTful APIs** *(3 Credits)*
*Description:* Design and build production-ready REST APIs — authentication, validation, error handling, and documentation.

**CS102 — Operating Systems & Networking** *(3 Credits)*
*Description:* OS fundamentals — processes, threads, memory, and networking — TCP/IP, DNS, HTTP/2.

**DB102 — Advanced SQL & Database Design** *(3 Credits)*
*Description:* Joins, subqueries, transactions, indexing strategies, normalization, and schema design.

**SEC101 — Introduction to Security** *(3 Credits)*
*Description:* Security fundamentals — OWASP Top 10, encryption, hashing, and secure coding practices.

**QA102 — Introduction to Testing** *(3 Credits)*
*Description:* Unit testing, integration testing, and test-driven development for backend applications.

## YEAR 200 — INTERMEDIATE BACKEND

### Semester 1

**DEV207 — Advanced Node.js (Streams, Child Processes)** *(3 Credits)*
*Description:* Advanced Node.js patterns — streams, buffers, child processes, clustering, and worker threads.

*Description:* Design microservice architectures — service boundaries, inter-service communication, and API gateways.

**DB201 — ORMs (Prisma, TypeORM)** *(3 Credits)*
*Description:* Use ORMs for database operations — Prisma and TypeORM with TypeScript, migrations, and relations.

### Semester 2

*Description:* Build GraphQL servers — schemas, resolvers, mutations, subscriptions, and federation.
