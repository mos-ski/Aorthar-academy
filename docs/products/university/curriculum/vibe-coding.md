*Quiz (5 questions):*
1. What is vibe coding? — **Answer:** Building software by describing requirements in natural language and using AI to generate, refine, and deploy code
2. What are the popular AI coding tools? — **Answer:** Cursor, Claude, GitHub Copilot, v0, Replit Agent, and Bolt.new
3. What is the prompt-to-code workflow? — **Answer:** Describe what you want → AI generates code → review and refine → iterate until it works → deploy
4. What is the biggest advantage of AI coding? — **Answer:** It removes the barrier of learning syntax — anyone with a clear idea can build software
5. What are the ethical considerations in AI coding? — **Answer:** Code quality, security vulnerabilities, attribution, over-reliance on AI, and understanding what the code actually does

*Quiz (5 questions):*
1. What makes a good code prompt? — **Answer:** Specific requirements, clear context, expected behavior, edge cases, and constraints
2. What is iterative prompting? — **Answer:** Starting with a basic prompt, reviewing the output, then refining with additional details — building up complexity gradually
3. How do you debug an AI-generated code prompt? — **Answer:** Describe the error, show the code, explain what you expected, and ask for a fix — be specific about the problem
4. What is a prompt template? — **Answer:** A reusable prompt structure for common tasks — e.g., "Build a [type] app that [functionality] using [tech stack]"
5. Why is context important in code prompts? — **Answer:** AI needs to understand the full picture — existing code, tech stack, constraints, and goals — to generate relevant code
