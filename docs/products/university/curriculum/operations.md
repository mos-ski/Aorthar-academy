*Quiz (5 questions):*
1. What are the four types of analytics? — **Answer:** Descriptive (what happened), Diagnostic (why it happened), Predictive (what will happen), Prescriptive (what should we do)
2. What is the difference between structured and unstructured data? — **Answer:** Structured data fits in tables with defined schemas; unstructured data (text, images, video) has no predefined format
3. What is the analytics lifecycle? — **Answer:** Define problem → collect data → clean data → analyze → visualize → communicate insights → act
4. Why is data privacy important in Nigeria? — **Answer:** NDPR regulates personal data handling; violations carry fines up to 2% of annual revenue; building user trust requires responsible data practices
5. What makes a decision data-driven? — **Answer:** It's based on evidence from data analysis, not intuition, tradition, or the highest-paid person's opinion

*Quiz (5 questions):*
1. What is the difference between mean, median, and mode? — **Answer:** Mean is the average; median is the middle value; mode is the most frequent value
2. What is a p-value? — **Answer:** The probability of observing results as extreme as yours if the null hypothesis is true — typically, p < 0.05 means the result is statistically significant
3. What is the normal distribution? — **Answer:** A bell-shaped distribution where most values cluster around the mean — many natural phenomena follow this pattern
4. What is the difference between correlation and causation? — **Answer:** Correlation means two variables move together; causation means one variable directly causes the other
5. What is a confidence interval? — **Answer:** A range of values that likely contains the true population parameter — e.g., "we're 95% confident the average is between 40 and 60"

*Quiz (5 questions):*
1. What is the difference between probability and non-probability sampling? — **Answer:** Probability: every member has a known chance of selection (random); non-probability: selection is not random (convenience, quota)
2. What makes a good survey question? — **Answer:** Clear, specific, unbiased, mutually exclusive options, and aligned with research objectives
3. What is sampling bias? — **Answer:** When the sample doesn't represent the population — e.g., surveying only smartphone users about mobile app usage
4. What is the minimum sample size for a survey? — **Answer:** Depends on population size, confidence level, and margin of error — typically 384 for a large population at 95% confidence, 5% margin
5. What is informed consent in data collection? — **Answer:** Participants must understand what data is collected, how it will be used, and voluntarily agree to participate

*Quiz (5 questions):*
1. What is the difference between WHERE and HAVING? — **Answer:** WHERE filters rows before grouping; HAVING filters groups after GROUP BY
2. What is a LEFT JOIN? — **Answer:** Returns all rows from the left table and matching rows from the right table — unmatched right rows are NULL
3. What is a CTE? — **Answer:** Common Table Expression — a named temporary result set defined with WITH clause, making complex queries more readable
4. What does a window function do? — **Answer:** Performs calculations across a set of rows related to the current row — without collapsing them like GROUP BY does
5. Why is SQL essential for data analysts? — **Answer:** Because most business data lives in relational databases, and SQL is the most efficient way to extract, filter, and aggregate it

*Quiz (5 questions):*
1. What is the purpose of linear regression? — **Answer:** To model the relationship between a dependent variable and one or more independent variables — predicting outcomes
2. What is ANOVA? — **Answer:** Analysis of Variance — testing whether the means of three or more groups are significantly different
3. What is a Type I vs Type II error? — **Answer:** Type I: false positive (rejecting true null); Type II: false negative (failing to reject false null)

5. What is A/B testing and how does it relate to statistics? — **Answer:** Comparing two versions to determine which performs better — using hypothesis testing to determine if the difference is statistically significant
