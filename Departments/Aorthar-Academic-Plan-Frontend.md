### **Course Deep Dive: DEV101 - JavaScript Fundamentals & ES6+**

**Description**: This course provides a comprehensive introduction to the JavaScript programming language, the language of the web. Students will go from basic syntax to modern ES6+ features, building a strong foundation for future framework development.

**12-Week Lesson Plan:**

| Week | Topic | Core Concepts | Resources |
|---|---|---|---|
| 1 | Intro to JS & Core Concepts | What is JS, adding it to a webpage, variables (`var`, `let`, `const`), data types. | JavaScript Tutorial for Beginners |
| 2 | Operators & Type Coercion | Arithmetic, assignment, comparison, logical operators; understanding `==` vs `===`. | JavaScript Operators |
| 3 | Control Flow: Conditionals | `if`, `else if`, `else` statements; ternary operator; `switch` statement. | If Else Statements |
| 4 | Control Flow: Loops | `for` loops, `while` loops, `do...while` loops, `for...in`, `for...of`. | JavaScript Loops Made Easy |
| 5 | Functions | Declaring functions, parameters, arguments, return values, function expressions. | JavaScript Functions Crash Course |
| 6 | Scope & Closures | Global vs. local scope, block scope, lexical scope, and the concept of closures. | Scope & Closures Explained |
| 7 | Arrays & Array Methods | Creating arrays, accessing elements, common methods (`push`, `pop`, `map`, `filter`). | JavaScript Array Methods |
| 8 | Objects | Object literals, properties, methods, bracket vs. dot notation, the `this` keyword. | JavaScript Objects |
| 9 | ES6+: Arrow Functions & Template Literals | New function syntax, lexical `this`, multi-line strings, and interpolation. | Arrow Functions |
| 10 | ES6+: Destructuring & Spread/Rest | Destructuring arrays and objects; spread syntax for arrays/objects; rest parameters. | Destructuring Explained |
| 11 | Asynchronous JS: Callbacks & Promises | The event loop, callback functions, callback hell, introduction to Promises. | Callbacks & Promises |
| 12 | Asynchronous JS: Async/Await | The `async` and `await` keywords for cleaner asynchronous code, error handling with `try...catch`. | Async Await |

**Sample Quiz Questions (Weeks 1-4):**

1.  **Which keyword is used to declare a variable that cannot be reassigned?**
    a) `var`
    b) `let`
    c) `const`
    d) `static`

2.  **What is the result of ` '5' == 5 ` in JavaScript?**
    a) `true`
    b) `false`
    c) `TypeError`
    d) `undefined`

3.  **Which statement is used to execute a block of code a specific number of times?**
    a) `if`
    b) `while`
    c) `switch`
    d) `for`

4.  **What does the `else` statement do in an `if...else` block?**
    a) It runs if the `if` condition is true.
    b) It always runs.
    c) It runs if the `if` condition is false.
    d) It defines a new condition to check.

**Final Exam Structure:**
- **Part A: Multiple Choice (40%)**: 40 questions on core concepts, syntax, and definitions.
- **Part B: Code Analysis (30%)**: 5 short JavaScript code snippets. Students must determine the output or identify the bug.
  - *Example*: "What will be logged to the console after this code runs? Explain your reasoning."
- **Part C: Practical Coding (30%)**: 2 small coding challenges to be solved in a simple editor.
  - *Example*: "Write a function that takes an array of numbers and returns a new array containing only the even numbers."
