# Module 07: Courses Catalog

## Route
- `/courses`

## Goal
- Present the curriculum list by year/semester/course for students.

## Key UX
- Year/level tabs and course cards.
- Open selected course in classroom mode.

## Data Sources
- `years`
- `semesters`
- `courses`
- `lessons`

## Access Rules
- Published courses only.
- Premium rules for protected levels/content.

## Admin Ownership
- Course metadata (code, title, description, pass mark, credit unit, status).
- Course publish/unpublish lifecycle.
